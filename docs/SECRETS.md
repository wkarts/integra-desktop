# Secrets do GitHub

## Obrigatórios para release

- `GITHUB_TOKEN` (fornecido automaticamente pelo GitHub Actions)

## Recomendados para assinatura/updater Tauri

- `TAURI_SIGNING_PRIVATE_KEY`
- `TAURI_SIGNING_PRIVATE_KEY_PASSWORD`

## Opcionais para Windows

- `WINDOWS_CERTIFICATE`
- `WINDOWS_CERTIFICATE_PASSWORD`

## Opcionais para macOS

- `APPLE_CERTIFICATE`
- `APPLE_CERTIFICATE_PASSWORD`
- `APPLE_ID`
- `APPLE_PASSWORD`
- `APPLE_TEAM_ID`
- `APPLE_API_KEY`
- `APPLE_API_ISSUER`
- `KEYCHAIN_PASSWORD`
